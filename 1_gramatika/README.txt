Gramatika pro jazyk
___________________________________
Základní operace a struktura
- sčítání "+"
- násobení "*"
- incementace "++ number"
- závorky "()"
Datove typy
- int, boolean, float
Unikátní vlastnost
- znak "&" bude oddělovat jednotlivá zadání pro kalkulačku, která budou řešena samostatně
